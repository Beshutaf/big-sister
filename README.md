Big Sister
==========

Mandatory data to save in the DB for each member:

- first name
- last name
- e-mail address
- physical address
- neighborhood
- phone number
- gender
- status (active/frozen/left)
- join date
- last status update date
- joint share with
- student status and degree
- age group
- role in shifts
- comments
- id
- birth date
- profession
- education
- skills
- civil status
- number of children

Extra data to be linked to other tables:

- shifts
- teams


The following API entry points shall be implemented:

- add member
- update member attribute
- get member attributes
- freeze member
- unfreeze member
- leave member
